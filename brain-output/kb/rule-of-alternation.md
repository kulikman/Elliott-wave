## Обязательные правила

1. **Для продолжения исследования группы волн в качестве возможного импульса между волнами 2 и 4 должно наблюдаться чередование значений как минимум одного из параметров: цена, время, сила отката, сложность или строение.** (AKU-0013) 🔴
> «Для продолжения исследования группы волн в качестве возможного импульса между волнами 2 и 4 должно наблюдаться чередование значений как минимум одного из описанных выше параметров.»
*Источник: neely-mwe-1990, стр. 5-3*

```
Когда: # Применяется при проверке группы волн как кандидата на импульс
pattern_candidate == 'impulse'
AND wave_2 IS defined
AND wave_4 IS defined
Правило: # Параметры чередования между волнами 2 и 4:

# A. Цена (длина волны в пунктах)
price_alternation = (price_length(wave_2) != price_length(wave_4))
# Существенное различие в абсолютной длине (амплитуде) волн 2 и 4

# B. Время (длительность волны)
time_alternation = (duration(wave_2) != duration(wave_4))
# Существенное различие в продолжительности волн 2 и 4

# C. Сила отката (процент коррекции предшествующей волны)
retracement_w2 = price_length(wave_2) / price_length(wave_1)
retracement_w4 = price_length(wave_4) / price_length(wave_3)
severity_alternation = (retracement_w2 != retracement_w4)
# Существенное различие в глубине отката волн 2 и 4

# D. Сложность (количество составляющих подволн)
complexity_alternation = (subwave_count(wave_2) != subwave_count(wave_4))
# Различие в количестве внутренних сегментов

# E. Строение (тип коррекционной фигуры)
construction_alternation = (pattern_type(wave_2) != pattern_type(wave_4))
# Например: wave_2 == 'zigzag' AND wave_4 == 'flat', или наоборот

# Обязательное условие: хотя бы один параметр должен чередоваться
alternation_exists = (
  price_alternation
  OR time_alternation
  OR severity_alternation
  OR complexity_alternation
  OR construction_alternation
)

# Если чередование отсутствует — группа волн НЕ может рассматриваться
# как возможный импульс (исследование в качестве импульса прекращается)
REQUIRE alternation_exists == true
```

2. **Волна-а и волна-b Зигзага должны отличаться по цене. Волна-b составит 61.8% или менее волны-а.** (AKU-0029) 🔴
> «Волна-а и волна-b Зигзага должны отличаться по цене. Волна-b составит 61.8% или менее волны-а.
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'zigzag'
Правило: length(wave_b) <= 0.618 * length(wave_a) AND length(wave_b) != length(wave_a)
```

3. **Импульсные фигуры должны иметь четко регистрируемые расширения (растяжения), и чередования должны происходить между второй и четвертой волной. Если два этих правила не выполняются, значит, формируется не Импульсная волна.** (AKU-0041) 🔴
> «Импульсные фигуры должны иметь четко регистрируемые расширения, и чередования должны происходить между второй и четвертой волной. Если два этих правила не выполняются, значит, формируется не Импульсна...»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse'
Правило: has_extension(impulse) == true AND alternation(wave_2, wave_4) == true
```

4. **Волна-1 не может сегментировать в отсутствие сегментированной волны-2 (если рынок не формирует Терминальную фигуру).** (AKU-0046) 🔴
> «волна-1 не можeт сегментировать в отсутствие сегментированной волны-2 (если рынок не формирует Терминальную фигуру).
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND subtype != 'terminal'
Правило: is_subdivided(wave_1) IMPLIES is_subdivided(wave_2)
```

5. **Когда волна-1 Импульса растянута, волна-2 должна быть сложнее волны-4.** (AKU-0048) 🔴
> «Когда волна-1 растянута, волна-2 должна быть сложнее волны-4.
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND extended_wave == 1
Правило: complexity(wave_2) > complexity(wave_4)
```

6. **Когда волна-5 Импульса растянута, волна-4 должна быть сложнее и длительнее волны-2.** (AKU-0049) 🔴
> «Когда волна-5 Растянутая, волна-4 должна быть сложнее и длительнее волны-2
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND extended_wave == 5
Правило: complexity(wave_4) > complexity(wave_2) AND duration(wave_4) > duration(wave_2)
```

7. **Если между волнами 2 и 4 нет никакого вида чередования (по времени, цене и структуре), то неважно, как выглядит данная фигура, — это движение не является Импульсным.** (AKU-0126) 🔴
> «Если между волнами 2 и 4 нет никакого вида чередования, то неважно, как выглядит данная фигура, - это движение не является Импульсным.
»
*Источник: neely-mwe-1990, стр. 12-50*

```
Когда: pattern == 'impulse'
Правило: alternation(wave_2, wave_4, by='time' OR 'price' OR 'structure') == true
```
