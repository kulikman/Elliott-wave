## Обязательные правила

1. **Если длина волны-2 слишком велика по сравнению с длиной волны-1, то волна-2 должна сегментировать, чтобы её волна-с не выходила за конечный уровень её волны-а; в противном случае большая волна начаться не сможет.** (AKU-0050) 🔴
> «Это указывает, что волна-2 должна сегментировать, чтобы ее волна-с не выходила за конечный уровень ее волны-а; в противном случае большая волна начаться не сможет.
»
*Источник: neely-mwe-1990, стр. 8-24*

```
Когда: pattern == 'impulse' AND length(wave_2) large_relative_to length(wave_1)
Правило: is_subdivided(wave_2) AND not exceeds(end(wave_2.c), end(wave_2.a))
```

2. **В трёх из четырёх ситуаций потери силы импульса (Неудавшаяся пятая волна, Неудавшаяся волна-С Плоской, завершение Сужающимся Неограничивающим Треугольником) точка экстремума (максимума или минимума) не является конечной точкой фигуры Эллиотта. Если следующая за фигурой волна достигла её начального уровня и самая высокая цена не является конечной точкой фигуры, то точка завершения будет возникать после достижения точки экстремума, а не до этого.** (AKU-0051) 🔴
> «B трех первых из четырех описанных ситуаций (пункты A, B и C) точка экстремума (максимума или минимума) не является конечной точкой фигуры Эллиотта.
»
*Источник: neely-mwe-1990, стр. 8-24*

```
Когда: subtype IN ('failure_5th', 'flat_failure_c', 'contracting_nonlimiting_triangle')
Правило: extreme_point(pattern) != end_point(pattern)
```

3. **Если волна-1 имеет уровень поливолны или выше, то волна-2 должна сегментировать на поливолновую или более высокого порядка фигуру.** (AKU-0085) 🔴
> «Если волна-1 имеет уровень поливолны или выше, то волна-2 должна сегментировать на поливолновую или более высокого порядка фигуру.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'impulse' AND degree(wave_1) >= 'polywave'
Правило: degree(wave_2) >= 'polywave'
```

4. **Когда волна «Пропавшая», она всегда представляет собой моноволну и никогда не бывает поливолной или более крупной фигурой.** (AKU-0122) 🔴
> «Когда волна "Пропавшая", она всегда представляет собой моноволну и никогда не бывает поливолной или более крупной фигурой.»
*Источник: neely-mwe-1990, стр. 12-45*

```
Когда: wave_status == 'missing'
Правило: degree(missing_wave) == 'monowave'
```

5. **Как только число точек данных достигает уровня вдвое больше минимального, пропажу волны следует считать невозможной.** (AKU-0123) 🔴
> «Как только число точек данных достигает уровня вдвое больше минимального, пропажу волны следует считать невозможной.»
*Источник: neely-mwe-1990, стр. 12-46*

```
Когда: evaluating missing_wave possibility
Правило: data_points >= 2 * minimum_data_points IMPLIES missing_wave_possible == false
```
