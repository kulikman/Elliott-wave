## Обязательные правила

1. **Волна 3 не может быть самой короткой из волн 1, 3 и 5. Если волна 3 короче обеих — импульсный счёт неверен.** (AKU-0002) 🔴
> «Волна 3 не может быть самой короткой волной в импульсной последовательности из волн 1, 3 и 5.
»
*Источник: neely-mwe-1990, стр. 3-15*

```
Когда: pattern == 'impulse'
Правило: length(wave_3) > min(length(wave_1), length(wave_5))
```

2. **Третий сегмент импульса никогда не может быть самым коротким из трёх трендовых сегментов (1, 3 и 5) при измерении вертикальных ценовых расстояний. Нарушение этого правила означает, что фигура не является импульсом.** (AKU-0007) 🔴
> «третий не обязательно должен быть самым длинным, но он никогда не может быть самым коротким из трех этих сегментов.»
*Источник: neely-mwe-1990, стр. 5-1*

```
Когда: # Правило применяется при проверке любой фигуры на импульсность
pattern == 'impulse'
Правило: # Измеряем вертикальные ценовые расстояния (длины) трёх трендовых сегментов
len_w1 = abs(price_end(wave_1) - price_start(wave_1))
len_w3 = abs(price_end(wave_3) - price_start(wave_3))
len_w5 = abs(price_end(wave_5) - price_start(wave_5))

# Третий сегмент никогда не может быть самым коротким из трёх
# Т.е. он должен быть строго больше хотя бы одного из двух других
# (эквивалентно: он НЕ является единственным минимумом или одним из минимумов)
NOT (len_w3 <= len_w1 AND len_w3 <= len_w5)

# Более явная запись:
# len_w3 > min(len_w1, len_w5)

# Если условие нарушено — фигура не является импульсом
```

3. **Для квалификации самой длинной волны импульсной группы в качестве Растянутой, её ценовая длина должна быть не меньше 161,8% второй по длине волны этой группы. Несоблюдение этого правила практически гарантирует коррективный характер анализируемой группы.** (AKU-0012) 🔴
> «Чтобы квалифицировать самую длинную волну в качестве Растянутой, ее ценовая длина должна быть не меньше 161,8% второй по длине волны этой группы. Несоблюдение любого из представленных до настоящего мо...»
*Источник: neely-mwe-1990, стр. 5-2*

```
Когда: # Правило применяется при проверке группы волн на импульсный характер
# Группа состоит из 5 волн (wave_1, wave_2, wave_3, wave_4, wave_5)
pattern_candidate == 'impulse'
Правило: # Вычисляем ценовые длины действующих (импульсных) волн группы
price_lengths = [
  price_length(wave_1),
  price_length(wave_3),
  price_length(wave_5)
]

# Сортируем по убыванию
sorted_lengths = sort_descending(price_lengths)

longest = sorted_lengths[0]   # самая длинная волна
second_longest = sorted_lengths[1]  # вторая по длине волна

# Самая длинная волна должна быть не меньше 161.8% второй по длине
longest >= second_longest * 1.618

# Если условие НЕ выполнено — группа практически гарантированно
# является коррективной, а не импульсной
# IF NOT (longest >= second_longest * 1.618):
#     pattern_candidate is likely corrective (fails impulse qualification)
```
