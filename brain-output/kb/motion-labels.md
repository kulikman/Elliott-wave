# Метки Движения — Knowledge Base

## Обязательные правила

1. **Волны 2, 4, b, d, e или x никогда не могут быть Импульсной фигурой. Импульс может быть обнаружен с Метками движения только в конкретных положениях, исключающих эти позиции.** (AKU-0079) 🔴
> «Волны 2, 4, b, d, e или x никогда не могут быть Импульсной фигурой.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: position IN ('wave_2','wave_4','wave_b','wave_d','wave_e','wave_x')
Правило: structure(wave) != ':5'
```

2. **В Трендовом Импульсе с Растянутой первой волной: движение после Растянутой волны-1 (волна-2) не может откатываться от волны-1 намного более 38.2%.** (AKU-0080) 🔴
> «Движение, следующее за Растянутой волной-1 (волна-2), не может откатываться от волны-1 намного более 38.2%.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'trend_impulse' AND extended_wave == 1
Правило: retrace(wave_2) <= 0.382 * length(wave_1)
```

3. **В Трендовом Импульсе с Растянутой первой волной: волна-2 не может быть Подвижной коррекцией.** (AKU-0081) 🔴
> «Волна-2 не может быть Подвижной коррекцией.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'trend_impulse' AND extended_wave == 1
Правило: type(wave_2) != 'moving_correction'
```

4. **В Трендовом Импульсе с Растянутой первой волной: волна-5 должна быть кратчайшей из трех импульсных волн (1, 3, 5).** (AKU-0082) 🔴
> «Волна-5 должна быть кратчайшей из трех импульсных волн (1, 3, 5).»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'trend_impulse' AND extended_wave == 1
Правило: length(wave_5) == min(length(wave_1), length(wave_3), length(wave_5))
```

5. **В Трендовом Импульсе с Нерастянутой волной-1: третья волна должна быть длиннее волны-1.** (AKU-0083) 🔴
> «Третья волна должна быть длиннее волны-1.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'trend_impulse' AND extended_wave != 1
Правило: length(wave_3) > length(wave_1)
```

6. **Если волна-1 является (или считается) самой длинной волной данной последовательности, то вторая волна не может откатиться более чем на 38.2% первой волны.** (AKU-0084) 🔴
> «Если волна-1 оказывается (или вы ее считаете) самой длинной волной данной последовательности, то вторая волна не может откатиться более чем на 38.2% первой волны.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'impulse' AND length(wave_1) == max(length(wave_1), length(wave_3), length(wave_5))
Правило: retrace(wave_2) <= 0.382 * length(wave_1)
```

7. **Если волна-1 имеет уровень поливолны или выше, то волна-2 должна сегментировать на поливолновую или более высокого порядка фигуру.** (AKU-0085) 🔴
> «Если волна-1 имеет уровень поливолны или выше, то волна-2 должна сегментировать на поливолновую или более высокого порядка фигуру.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'impulse' AND degree(wave_1) >= 'polywave'
Правило: degree(wave_2) >= 'polywave'
```

8. **Никакая Коррективная фигура не может появиться в любом положении Метки движения Трендового Импульса (т.е. на месте волн 1, 3, 5, волны-а Зигзага или волны-с Зигзага или Плоской всех вариаций).** (AKU-0086) 🔴
> «никакая Коррективная фигура не может появиться в любом положении Метки движения Трендового Импульса (т. e., на месте волн 1, 3, 5, волны-а Зигзага или волны-с Зигзага или Плоской [всех вариаций]).»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: position IN ('wave_1','wave_3','wave_5','zigzag.wave_a','zigzag.wave_c','flat.wave_c')
Правило: structure(wave) == ':5'
```

9. **В Терминальном Импульсе с Растянутой первой волной: волна-2 не должна откатываться более чем на 61.8% волны-1. Волна-3 не должна быть намного больше 61.8%, но и не менее 38.2% волны-1. Волна-5 не может быть более 99% волны-3.** (AKU-0087) 🔴
> «волна-2 не должна откатываться более чем на 61.8% волны-1. Волна-3 не должна быть намного больше 61.8%, но и не менее 38.2% волны-1. Волна-5 не может быть более 99% волны-3»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'terminal_impulse' AND extended_wave == 1
Правило: retrace(wave_2) <= 0.618 * length(wave_1) AND 0.382 * length(wave_1) <= length(wave_3) <= 0.618 * length(wave_1) AND length(wave_5) <= 0.99 * length(wave_3)
```

10. **В Обыкновенной Плоской фигуре волна-b не должна быть более 100% волны-а, но должна откатываться как минимум на 81% волны-а, и волна-с должна откатываться на всю величину волны-b. Волна-с должна быть Импульсной.** (AKU-0088) 🔴
> «Волна-Ь не должна быть более 100% волны-а, но должна откатываться как минимум на 81% волны-а, и волна-с должна откатываться на всю величину волны-Ь.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'flat' AND subtype == 'common'
Правило: 0.81 * length(wave_a) <= length(wave_b) <= 1.00 * length(wave_a) AND length(wave_c) >= length(wave_b) AND structure(wave_c) == ':5'
```

11. **Ни при каких обстоятельствах волна-d не может быть крупнейшей волной любой Треугольной фигуры.** (AKU-0089) 🔴
> «ни при каких обстоятельствах волна-d не может быть крупнейшей волной любой Треугольной фигуры.
»
*Источник: neely-mwe-1990, стр. 11-19*

```
Когда: pattern == 'triangle'
Правило: length(wave_d) != max(length(a), length(b), length(c), length(d), length(e))
```

12. **В Неправильной Неудавшейся Плоской коррекции волна-с не должна откатываться до начального уровня волны-b. Волна-b должна быть больше волны-а в ценовом отношении.** (AKU-0094) 🔴
> «Волна-b должна быть больше волны-а в ценовом отношении. Волна-с не должна откатываться до начального уровня волны-b.
»
*Источник: neely-mwe-1990, стр. 11-18*

```
Когда: pattern == 'flat' AND subtype == 'irregular_failure'
Правило: not reaches(end(wave_c), start_level(wave_b)) AND length(wave_b) > length(wave_a)
```

13. **Движение непосредственно после Подвижной коррекции должно всегда быть самым длинным сегментом завершённой фигуры Эллиотта.** (AKU-0095) 🔴
> «Движение непосредственно после Подвижной коррекции должно всегда быть самым длинным сегментом завершенной фигуры Эллиотта.
»
*Источник: neely-mwe-1990, стр. 11-18*

```
Когда: pattern == 'moving_correction' AND position == 'post_pattern'
Правило: length(next_segment) == max(length(all_segments(completed_pattern)))
```

14. **В Зигзаге волна-а должна быть Импульсной, и волна-b не должна откатиться от неё более чем на 61.8%. Волна-b однозначно должна быть коррективной и не должна откатываться от волны-а более чем на 61.8%.** (AKU-0096) 🔴
> «Волна-а должна быть Импульсной и волна-Ь не должна откатиться от нее более чем на 61.8%... Волна-b однозначно должна быть коррективной (3) и не должна откатываться от волны-а более чем на 61,8%
»
*Источник: neely-mwe-1990, стр. 11-20*

```
Когда: pattern == 'zigzag'
Правило: structure(wave_a) == ':5' AND structure(wave_b) == ':3' AND retrace(wave_b) <= 0.618 * length(wave_a)
```

15. **Волна-b Зигзага не должна быть Подвижной Коррекцией, за исключением случая, когда Зигзаг входит в состав Треугольника.** (AKU-0097) 🔴
> «Она не должна быть Подвижной Коррекцией, не считая случая, когда Зигзаг, частью которого может быть данная b-волна, входит в состав Треугольника.
»
*Источник: neely-mwe-1990, стр. 11-20*

```
Когда: pattern == 'zigzag' AND parent != 'triangle'
Правило: type(wave_b) != 'moving_correction'
```

16. **В Горизонтальном Сужающемся Ограничивающем Треугольнике волна-d должна быть короче волны-с, и волна-е должна быть короче волны-d. Волна-b не должна быть менее 38.2% или более 261.8% волны-а.** (AKU-0098) 🔴
> «Волна-d должна быть короче волны-с и волна-е должна быть короче волны-d... волна-Ь не должна быть менее 38.2% или более 261.8% волны - а.
»
*Источник: neely-mwe-1990, стр. 11-22*

```
Когда: pattern == 'horizontal_contracting_limiting_triangle'
Правило: length(wave_d) < length(wave_c) AND length(wave_e) < length(wave_d) AND 0.382 * length(wave_a) <= length(wave_b) <= 2.618 * length(wave_a)
```

17. **Любой Неограничивающий Треугольник (за исключением случая, когда он х-волна) должен завершать фазу более крупной Коррекции. Если Неограничивающий Треугольник обнаружен в какой-либо позиции (кроме позиции x-волны), то всё более крупное движение можно сжать (уплотнить) до простой тройки.** (AKU-0099) 🔴
> «любой Неограничивающий Треугольник (за исключением случая, когда он х-волна) должен завершать фазу более крупной Коррекции. Если вы обнаружили один из них в какой-либо позиции (кроме позиции x-волны),...»
*Источник: neely-mwe-1990, стр. 11-27*

```
Когда: pattern == 'nonlimiting_triangle' AND role != 'x_wave'
Правило: completes_phase(larger_correction) == true
```

18. **Расширяющиеся Треугольники не могут иметь место непосредственно перед любой мощной фигурой, такой как Растянутая 1-я, 2-я или 3-я волна либо Удлинённая c-волна. Они также не могут завершать Двойную или Тройную Коррекцию или Комбинацию Коррекций Подвижная Тройка.** (AKU-0100) 🔴
> «Расширяющиеся Треугольники не могут иметь место непосредственно перед любой мощной фигурой, такой как Растянутая 1-я, 2-я или 3-я волна либо Удлиненная c-волна. Они не могут завершать двойную или Трой...»
*Источник: neely-mwe-1990, стр. 11-31*

```
Когда: pattern == 'expanding_triangle'
Правило: next_pattern NOT IN ('extended_1','extended_2','extended_3','elongated_c') AND not ends_correction(['double','triple','combination'])
```

19. **Фигура после Расширяющегося Треугольника (волна-с или волна-5) не должна откатываться до начального уровня волны-е.** (AKU-0101) 🔴
> «Фигура после Треугольника (волна-с или волна-5) не должна откатываться до начального уровня волны-е.
»
*Источник: neely-mwe-1990, стр. 11-31*

```
Когда: pattern == 'expanding_triangle' AND position == 'post_pattern' AND role IN ('wave_c','wave_5')
Правило: not reaches(next_wave, start_level(wave_e))
```

20. **Расширяющийся Ограничивающий Треугольник в позиции b-волны может быть только частью Плоской коррекции. Он не может быть b-волной Зигзага, x-волной Сложной Коррекции или b-волной более крупного Треугольника.** (AKU-0102) 🔴
> «Если он b-волна, то может быть только частью Плоской. Этот Треугольник, по всей видимости, не может быть Ь-волной Зигзага, x-волной Сложной Коррекции или Ь-волной более крупного Треугольника.
»
*Источник: neely-mwe-1990, стр. 11-31*

```
Когда: pattern == 'expanding_limiting_triangle' AND position == 'wave_b'
Правило: parent == 'flat' AND parent NOT IN ('zigzag', 'complex_correction_x', 'larger_triangle')
```

21. **В Расширяющемся Ограничивающем Горизонтальном Треугольнике волна-e должна быть самой интенсивной, сложной и длительной в группе.** (AKU-0105) 🔴
> «Волна-e должна быть самой интенсивной, сложной и длительной в группе.
»
*Источник: neely-mwe-1990, стр. 11-31*

```
Когда: pattern == 'expanding_limiting_horizontal_triangle'
Правило: complexity(wave_e) == max AND duration(wave_e) == max AND length(wave_e) == max
```
