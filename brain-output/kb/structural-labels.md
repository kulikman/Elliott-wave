## Обязательные правила

1. **В Треугольнике ровно пять сегментов — ни больше, ни меньше. Каждому сегменту присваивается буквенное обозначение в порядке появления: a, b, c, d, e.** (AKU-0018) 🔴
> «Эллиотт говорил, что в Треугольнике пять сегментов - ни больше, ни меньше. B соответствии c порядком появления, каждому сегменту Треугольника присваивается буквенное обозначение: (а, b, c, d, e).
»
*Источник: neely-mwe-1990, стр. 5-29*

```
Когда: pattern == 'triangle'
Правило: segment_count == 5 AND labels(segments) == ['a','b','c','d','e']
```

2. **Каждый сегмент Треугольника является Полной Коррективной фазой (тройкой, ":3").** (AKU-0019) 🔴
> «Каждый сегмент Треугольника - Полная Коррективная фаза (тройка, ":3")
»
*Источник: neely-mwe-1990, стр. 5-29*

```
Когда: pattern == 'triangle'
Правило: for seg in [a,b,c,d,e]: structure(seg) == ':3'
```

3. **Необходимо вовремя идентифицировать и классифицировать (по признаку «Импульс — Коррекция», т.е. «:5» или «:3») все составляющие каждого сегмента каждой волны графика. Необходимо придерживаться предыдущей, проверенной ценовой структуры, даже если намерения рынка не вполне ясны.** (AKU-0058) 🔴
> «Если вовремя не идентифицировать и не классифицировать (по признаку "Импульс - Коррекция", т. е. ":5" или ":3") все составляющие каждого сегмента каждой волны графика, то с увеличением исследуемой фиг...»
*Источник: neely-mwe-1990, стр. 9-6*

4. **Никогда ни одна из сторон Сужающегося Треугольника (кроме волны-е) не может быть Сужающимся Треугольником.** (AKU-0091) 🔴
> «Никогда ни одна из сторон Сужающегося Треугольника (кроме волны-е) не может быть Сужающимся Треугольником.
»
*Источник: neely-mwe-1990, стр. 11-19*

```
Когда: pattern == 'contracting_triangle'
Правило: for seg in [a,b,c,d]: type(seg) != 'contracting_triangle'
```

5. **Волны а, b, c или d Треугольника никогда не могут сами быть Треугольниками. Волна-е — единственный сегмент в Треугольнике, который может быть меньшим Треугольником.** (AKU-0092) 🔴
> «волны а, b, c или d Треугольника никогда не могут сами быть Треугольниками. Волна-е - единственный сегмент в Треугольнике, который может быть меньшим Треугольником.
»
*Источник: neely-mwe-1990, стр. 11-25*

```
Когда: pattern == 'triangle'
Правило: for seg in [a,b,c,d]: type(seg) != 'triangle'
```
