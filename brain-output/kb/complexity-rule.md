## Обязательные правила

1. **Импульсные фигуры должны иметь четко регистрируемые расширения (растяжения), и чередования должны происходить между второй и четвертой волной. Если два этих правила не выполняются, значит, формируется не Импульсная волна.** (AKU-0041) 🔴
> «Импульсные фигуры должны иметь четко регистрируемые расширения, и чередования должны происходить между второй и четвертой волной. Если два этих правила не выполняются, значит, формируется не Импульсна...»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse'
Правило: has_extension(impulse) == true AND alternation(wave_2, wave_4) == true
```

2. **Ни один из нечетных сегментов импульсной фигуры не должен сегментировать раньше одного из сегментов коррективных фигур (если импульсная фигура не терминальная).** (AKU-0045) 🔴
> «Ни один из нечетных сегментов импульсной фигуры не должен "делиться" раньше одного из сегментов коррективных фигур (если импульсная фигура не терминальная).
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND subtype != 'terminal'
Правило: not (any odd_wave in [1,3,5] is_subdivided WHILE all corrective_waves in [2,4] are monowaves)
```

3. **Волна-1 не может сегментировать в отсутствие сегментированной волны-2 (если рынок не формирует Терминальную фигуру).** (AKU-0046) 🔴
> «волна-1 не можeт сегментировать в отсутствие сегментированной волны-2 (если рынок не формирует Терминальную фигуру).
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND subtype != 'terminal'
Правило: is_subdivided(wave_1) IMPLIES is_subdivided(wave_2)
```

4. **Уровень сложности x-волны должен как минимум совпадать с уровнем непосредственно ей предшествующей c-волны и не должен превышать уровень самой сложной Стандартной фигуры в завершенной конфигурации.** (AKU-0047) 🔴
> «Уровень сложности x-волны должен кaк минимум совпадать c уровнем непосредственно ей предшествующей c-волны и не должен пpeвышать уровень самой сложной Стандартной фигyры в завершенной конфигурации.
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'x-wave'
Правило: complexity(x_wave) >= complexity(preceding_c_wave) AND complexity(x_wave) <= max_complexity(standard_patterns_in_config)
```

5. **Когда волна-1 Импульса растянута, волна-2 должна быть сложнее волны-4.** (AKU-0048) 🔴
> «Когда волна-1 растянута, волна-2 должна быть сложнее волны-4.
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND extended_wave == 1
Правило: complexity(wave_2) > complexity(wave_4)
```

6. **Когда волна-5 Импульса растянута, волна-4 должна быть сложнее и длительнее волны-2.** (AKU-0049) 🔴
> «Когда волна-5 Растянутая, волна-4 должна быть сложнее и длительнее волны-2
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND extended_wave == 5
Правило: complexity(wave_4) > complexity(wave_2) AND duration(wave_4) > duration(wave_2)
```
