## Обязательные правила

1. **В трёх из четырёх ситуаций потери силы импульса (Неудавшаяся пятая волна, Неудавшаяся волна-С Плоской, завершение Сужающимся Неограничивающим Треугольником) точка экстремума (максимума или минимума) не является конечной точкой фигуры Эллиотта. Если следующая за фигурой волна достигла её начального уровня и самая высокая цена не является конечной точкой фигуры, то точка завершения будет возникать после достижения точки экстремума, а не до этого.** (AKU-0051) 🔴
> «B трех первых из четырех описанных ситуаций (пункты A, B и C) точка экстремума (максимума или минимума) не является конечной точкой фигуры Эллиотта.
»
*Источник: neely-mwe-1990, стр. 8-24*

```
Когда: subtype IN ('failure_5th', 'flat_failure_c', 'contracting_nonlimiting_triangle')
Правило: extreme_point(pattern) != end_point(pattern)
```

2. **Ни одна часть волны-3 не должна пробивать настоящую линию тренда 0-2 в Импульсной фигуре. Если повышение после предполагаемой волны-2 не имеет силы остаться выше линии тренда 0-2, то оно по своему характеру не Импульсное и не может быть частью Импульсной волны-3.** (AKU-0107) 🔴
> «ни одна часть волны-3 не должна пробивать настоящую линию тренда 0-2. Если повышение, удаляющееся от волны-2, не имеет силы остаться выше "первоначальной" линии тренда 0-2, то оно по своему характеру ...»
*Источник: neely-mwe-1990, стр. 12-2*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(start(0), end(2)))
```

3. **Ни одна часть волны-3 или волны-5 не должна пробивать линию тренда 2-4, если только 5-я волна не является Терминальной фигурой.** (AKU-0108) 🔴
> «ни одна часть волны-З или волны-5 не должна пробивать линию тренда 2-4, если только 5-я волна не является Терминальной фигурой.
»
*Источник: neely-mwe-1990, стр. 12-4*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(end(2), end(4))) AND (not breaks(wave_5, trendline(end(2), end(4))) OR subtype == 'terminal')
```

4. **Построение линии тренда 0-B через волну-е Треугольника (когда волна-b является Треугольником) недопустимо, если это приведёт к её пробитию волной-с. Линию тренда следует проводить через волну-с Треугольника, а не волну-е.** (AKU-0111) 🔴
> «линия тренда "О-В" фактически проведена через волну-с Треугольника, а не волну-е. Построение линии тренда по волне-е привело бы к ее пробитию волной-с, что недопустимо.
»
*Источник: neely-mwe-1990, стр. 12-5*

```
Когда: position == 'wave_b' AND type(wave_b) == 'triangle'
Правило: trendline(0, B) drawn_through end(wave_c) NOT end(wave_e)
```

5. **Импульсная волна не может быть сформирована всего из четырёх моноволн, она должна содержать по меньшей мере пять (5) моноволн. Для Коррекции необходимы как минимум три (3) моноволны.** (AKU-0121) 🔴
> «Импульсная волна не может быть сформирована всего из четырёх моноволн, она должна содержать, по меньшей мере, пять (5). Для Коррекции необходимы как минимум три (3) моноволны.»
*Источник: neely-mwe-1990, стр. 12-45*

```
Когда: classifying pattern
Правило: (is_impulse IMPLIES monowave_count >= 5) AND (is_correction IMPLIES monowave_count >= 3)
```

6. **Когда волна «Пропавшая», она всегда представляет собой моноволну и никогда не бывает поливолной или более крупной фигурой.** (AKU-0122) 🔴
> «Когда волна "Пропавшая", она всегда представляет собой моноволну и никогда не бывает поливолной или более крупной фигурой.»
*Источник: neely-mwe-1990, стр. 12-45*

```
Когда: wave_status == 'missing'
Правило: degree(missing_wave) == 'monowave'
```

7. **Как только число точек данных достигает уровня вдвое больше минимального, пропажу волны следует считать невозможной.** (AKU-0123) 🔴
> «Как только число точек данных достигает уровня вдвое больше минимального, пропажу волны следует считать невозможной.»
*Источник: neely-mwe-1990, стр. 12-46*

```
Когда: evaluating missing_wave possibility
Правило: data_points >= 2 * minimum_data_points IMPLIES missing_wave_possible == false
```

8. **Нечётные сегменты (волны 3 и 5) Импульсной волны не могут представлять собой Зигзаги вместо Импульсов. Такой тип развития нечётных сегментов Импульсной волны невозможен, если только рынок не формирует Терминальную (конечный диагональный треугольник) фигуру.** (AKU-0129) 🔴
> «волны 3 и 5 явным образом представляют собой Зигзаги, а не Импульсы. Такой тип развития нечетных сегментов Импульсной волны невозможен, если только рынок не формирует Терминальную фигуру.
»
*Источник: neely-mwe-1990, стр. 12-53*

```
Когда: pattern == 'impulse' AND subtype != 'terminal'
Правило: structure(wave_3) != 'zigzag' AND structure(wave_5) != 'zigzag'
```

9. **Когда в Импульсной фигуре растягивается первая волна, волна-2 должна почти всегда быть самой сложной и длительной по времени фигурой.** (AKU-0132) 🔴
> «Когда растягивается первая волна, волна-2 должна почти всегда быть самой сложной и длительной по времени фигурой.
»
*Источник: neely-mwe-1990, стр. 12-54*

```
Когда: pattern == 'impulse' AND extended_wave == 1
Правило: complexity(wave_2) == max AND duration(wave_2) == max
```
