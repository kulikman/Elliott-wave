# Плоская коррекция — Knowledge Base

> Топик: `flat` | AKU verified: 19 | Источники: neely-mwe-1990: 19
> Сгенерировано: 2026-06-02 18:54 UTC

## Обязательные правила

1. **В Плоской коррекции, когда волна-b превышает 138.2% волны-а, нет ни малейшего шанса, что волна-с превысит начальный уровень волны-b (это возможно в случае с-волны Треугольника, но не в случае с-волны Плоской).** (AKU-0017) 🔴
> «Когда волна-Ь превышает 138.2% волны-а, нет ни малейшего шанса, что волна-с превысит начальный уровень волны-Ь (это возможно в случае с-волны Треугольника, но не в случае с-волны Плоской).
»
*Источник: neely-mwe-1990, стр. 5-21*

```
Когда: pattern == 'flat' AND length(wave_b) > 1.382 * length(wave_a)
Правило: not exceeds(end(wave_c), start_level(wave_b))
```

2. **В Плоской коррекции с нормальной b-волной, волна-с не может составлять менее 38.2% волны-а — это минимальное требование, ниже которого она опуститься не может.** (AKU-0025) 🔴
> «Очень редко волна-с может соотноситься с волной-а с коэффициентом 38.2%, но это минимальное требование (ниже которого она опуститься не может).
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'flat' AND b_type == 'normal'
Правило: length(wave_c) >= 0.382 * length(wave_a)
```

3. **В Плоской коррекции с сильной b-волной (превышающей начальный уровень волны-а), b-волна обычно ограничена соотношением 138.2% или 161.8% волны-а, но не может достигать ни одного из них.** (AKU-0026) 🔴
> «Когда b-волна превышает начальный уровень волны-а, обычно она будет ограничена соотношением 138.2% (это ненастоящее соотношение Фибоначчи, но представляет комбинацию двух - 1.00 и 0.382) или 161.8%, н...»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'flat' AND end_level(wave_b) beyond start_level(wave_a)
Правило: length(wave_b) < 1.618 * length(wave_a) AND length(wave_b) != 1.382 * length(wave_a) AND length(wave_b) != 1.618 * length(wave_a)
```

4. **Для Плоских и Зигзагов, где волна-b короче волны-a: следующая за Коррекцией волна должна пробить линию тренда 0-B за период времени, не превышающий длительности волны-с. Если этот процесс занимает больше времени, то либо волна-с формирует Терминал, либо четвёртая волна (волны-с) ещё не завершилась, либо интерпретация коррекции ошибочная.** (AKU-0036) 🔴
> «следующая за данной Коррекцией волна должна пробить линию тренда 0-В за период времени, не превышающий длительности волны-с. Если этот процесс занимает больше времени, то либо волна-с формирует Термин...»
*Источник: neely-mwe-1990, стр. 6-2*

```
Когда: pattern IN ('flat','zigzag') AND length(wave_b) < length(wave_a) AND position == 'post_pattern'
Правило: duration(until_break(trendline(start(0), end(B)))) <= duration(wave_c)
```

5. **Первая после окончания Подвижной Коррекции волна должна быть взрывной (повышательной или понижательной) и довольно длинной — не меньше 161,8% волны (или группы волн), предшествовавшей этой Подвижной Коррекции.** (AKU-0040) 🔴
> «первая после окончания Подвижной Коррекции волна должна быть взрывной (повышательной или понижательной) и довольно длинной - не меньше 161,8% волны (или группы волн), предшествовавшей этой Подвижной К...»
*Источник: neely-mwe-1990, стр. 6-3*

```
Когда: pattern == 'moving_correction' AND position == 'post_pattern'
Правило: length(next_wave) >= 1.618 * length(preceding_wave_or_group)
```

6. **Следующая за Неудавшейся-с волна такого же Порядка должна достигать её начального уровня.** (AKU-0064) 🔴
> «Следующая за Неудавшейся-с волна такого же Порядка должна достигать ее начального уровня.
»
*Источник: neely-mwe-1990, стр. 10-5*

```
Когда: subtype == 'failure_c' AND position == 'post_pattern'
Правило: reaches(next_wave_same_degree, start_level(pattern))
```

7. **Следующая за Неправильной Неудавшейся волна такого же Порядка должна достигать её начального уровня.** (AKU-0065) 🔴
> «Следующая за Неправильной Неудавшейся волна такого же Порядка должна достигать ее начального уровня.
»
*Источник: neely-mwe-1990, стр. 10-5*

```
Когда: subtype == 'irregular_failure' AND position == 'post_pattern'
Правило: reaches(next_wave_same_degree, start_level(pattern))
```

8. **Если Расширяющийся Треугольник возникает в позиции b-волны, Неудавшаяся-с неизбежна.** (AKU-0070) 🔴
> «если данная фигура возникает в позиции b-волны, Неудавшаяся-с неизбежна.
»
*Источник: neely-mwe-1990, стр. doc-ch10*

```
Когда: pattern == 'expanding_triangle' AND position == 'wave_b'
Правило: next_structure_contains('failure_c') == true
```

9. **Никакая Коррективная фигура не может появиться в любом положении Метки движения Трендового Импульса (т.е. на месте волн 1, 3, 5, волны-а Зигзага или волны-с Зигзага или Плоской всех вариаций).** (AKU-0086) 🔴
> «никакая Коррективная фигура не может появиться в любом положении Метки движения Трендового Импульса (т. e., на месте волн 1, 3, 5, волны-а Зигзага или волны-с Зигзага или Плоской [всех вариаций]).»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: position IN ('wave_1','wave_3','wave_5','zigzag.wave_a','zigzag.wave_c','flat.wave_c')
Правило: structure(wave) == ':5'
```

10. **В Обыкновенной Плоской фигуре волна-b не должна быть более 100% волны-а, но должна откатываться как минимум на 81% волны-а, и волна-с должна откатываться на всю величину волны-b. Волна-с должна быть Импульсной.** (AKU-0088) 🔴
> «Волна-Ь не должна быть более 100% волны-а, но должна откатываться как минимум на 81% волны-а, и волна-с должна откатываться на всю величину волны-Ь.»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'flat' AND subtype == 'common'
Правило: 0.81 * length(wave_a) <= length(wave_b) <= 1.00 * length(wave_a) AND length(wave_c) >= length(wave_b) AND structure(wave_c) == ':5'
```

11. **В Неправильной Неудавшейся Плоской коррекции волна-с не должна откатываться до начального уровня волны-b. Волна-b должна быть больше волны-а в ценовом отношении.** (AKU-0094) 🔴
> «Волна-b должна быть больше волны-а в ценовом отношении. Волна-с не должна откатываться до начального уровня волны-b.
»
*Источник: neely-mwe-1990, стр. 11-18*

```
Когда: pattern == 'flat' AND subtype == 'irregular_failure'
Правило: not reaches(end(wave_c), start_level(wave_b)) AND length(wave_b) > length(wave_a)
```

12. **Движение непосредственно после Подвижной коррекции должно всегда быть самым длинным сегментом завершённой фигуры Эллиотта.** (AKU-0095) 🔴
> «Движение непосредственно после Подвижной коррекции должно всегда быть самым длинным сегментом завершенной фигуры Эллиотта.
»
*Источник: neely-mwe-1990, стр. 11-18*

```
Когда: pattern == 'moving_correction' AND position == 'post_pattern'
Правило: length(next_segment) == max(length(all_segments(completed_pattern)))
```

13. **Волна-b Зигзага не должна быть Подвижной Коррекцией, за исключением случая, когда Зигзаг входит в состав Треугольника.** (AKU-0097) 🔴
> «Она не должна быть Подвижной Коррекцией, не считая случая, когда Зигзаг, частью которого может быть данная b-волна, входит в состав Треугольника.
»
*Источник: neely-mwe-1990, стр. 11-20*

```
Когда: pattern == 'zigzag' AND parent != 'triangle'
Правило: type(wave_b) != 'moving_correction'
```

14. **Расширяющийся Ограничивающий Треугольник в позиции b-волны может быть только частью Плоской коррекции. Он не может быть b-волной Зигзага, x-волной Сложной Коррекции или b-волной более крупного Треугольника.** (AKU-0102) 🔴
> «Если он b-волна, то может быть только частью Плоской. Этот Треугольник, по всей видимости, не может быть Ь-волной Зигзага, x-волной Сложной Коррекции или Ь-волной более крупного Треугольника.
»
*Источник: neely-mwe-1990, стр. 11-31*

```
Когда: pattern == 'expanding_limiting_triangle' AND position == 'wave_b'
Правило: parent == 'flat' AND parent NOT IN ('zigzag', 'complex_correction_x', 'larger_triangle')
```

15. **Построение линии тренда 0-B через волну-е Треугольника (когда волна-b является Треугольником) недопустимо, если это приведёт к её пробитию волной-с. Линию тренда следует проводить через волну-с Треугольника, а не волну-е.** (AKU-0111) 🔴
> «линия тренда "О-В" фактически проведена через волну-с Треугольника, а не волну-е. Построение линии тренда по волне-е привело бы к ее пробитию волной-с, что недопустимо.
»
*Источник: neely-mwe-1990, стр. 12-5*

```
Когда: position == 'wave_b' AND type(wave_b) == 'triangle'
Правило: trendline(0, B) drawn_through end(wave_c) NOT end(wave_e)
```

16. **Волна-е при Треугольной волне-b не должна превышать концевую волну-с Треугольника, даже при «ложном» пробитии линии тренда 0-B. «Ложное» пробитие должно быть очень краткосрочным (относительно).** (AKU-0112) 🔴
> «Это "ложное" пробитые должно быть очень краткосрочным (относительно), и волна-е не должна превышать концевую волну-с Треугольника.
»
*Источник: neely-mwe-1990, стр. 12-6*

```
Когда: position == 'wave_b' AND type(wave_b) == 'triangle'
Правило: not exceeds(end(wave_e), end(wave_c))
```

17. **C-волна Плоской не должна превышать 138.2% а-волны. Если с-волна больше 138.2% волны-а, эта фигура попадает в категорию Удлинённой Плоской.** (AKU-0119) 🔴
> «C-волна Плоской не должна превышать 138.2% а-волны»
*Источник: neely-mwe-1990, стр. 12-31*

```
Когда: pattern == 'flat'
Правило: length(wave_c) <= 1.382 * length(wave_a)
```

18. **Основная разница между двойной Плоской и двойным Зигзагом в том, что волна-b в первом Зигзаге не будет откатываться от волны-а более 61.8%. Настоящие 2-е волны не должны завершаться в точке, представляющей откат волны-1 более чем на 61.8%.** (AKU-0133) 🔴
> «настоящие 2-e волны не должны завершаться в точке, представляющей откат волны-1 более чем на 61.8%.
»
*Источник: neely-mwe-1990, стр. 12-51*

```
Когда: pattern == 'double_zigzag'
Правило: retrace(wave_b of first_zigzag) <= 0.618 * length(wave_a)
```

## Условные правила

1. **Все Стандартные Коррекции (кроме Треугольника) характеризуются четырьмя возможными точками касания, но только три из них могут действительно касаться параллельных линий тренда. Если четыре точки одинакового Порядка касаются параллельных линий тренда Стандартной Коррекции (за исключением Треугольника), значит, эта Коррекция — часть Сложной Нестандартной конфигурации.** (AKU-0055) 🟡
> «Все Стандартные Коррекции характеризуются четырьмя возможными точками касания, но только три из них могут действительно касаться параллельных линий тренда. Если четыре точки одинакового Порядка касают...»
*Источник: neely-mwe-1990, стр. 9-2*

```
Когда: pattern IN corrective AND pattern != 'triangle'
Правило: count(same_degree_points touching parallel_trendlines) <= 3
```
