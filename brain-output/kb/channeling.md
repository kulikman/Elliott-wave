## Обязательные правила

1. **Ни одна часть волн 1 или 3 не должна пересекать действительную линию тренда 0-2.** (AKU-0011) 🔴
> «ни одна часть волн 1 или 3 не должна пересекать действительную линию тренда 0-2.»
*Источник: neely-mwe-1990, стр. 5-6*

```
Когда: # Применяется к импульсной фигуре, когда определены точки 0, 1, 2, 3
# и можно построить линию тренда через точки 0 и 2
pattern == 'impulse'
AND wave_0 IS defined
AND wave_1 IS defined
AND wave_2 IS defined
AND wave_3 IS defined
Правило: # Построить действительную линию тренда (base channel line) через точки 0 и 2
# Точка 0 = начало волны 1, Точка 2 = конец волны 2
trendline_0_2 = line_through(point_0, point_2)

# Для каждого ценового бара внутри волны 1:
#   ни один бар не должен пересекать (пробивать) линию тренда 0-2
FOR each bar IN wave_1.all_bars:
  IF direction == 'up':
    # В бычьем импульсе линия 0-2 проходит по низам, 
    # ни один low бара волны 1 не должен быть ниже значения линии тренда в этот момент
    bar.low >= trendline_0_2.value_at(bar.time)
  ELIF direction == 'down':
    # В медвежьем импульсе линия 0-2 проходит по верхам,
    # ни один high бара волны 1 не должен быть выше значения линии тренда в этот момент
    bar.high <= trendline_0_2.value_at(bar.time)

# Для каждого ценового бара внутри волны 3:
#   ни один бар не должен пересекать (пробивать) линию тренда 0-2
FOR each bar IN wave_3.all_bars:
  IF direction == 'up':
    bar.low >= trendline_0_2.value_at(bar.time)
  ELIF direction == 'down':
    bar.high <= trendline_0_2.value_at(bar.time)
```

2. **Линия канала, пересекающая волны b и d Треугольника, является Базовой линией. В качестве общего правила, трендовая линия b-d не должна пробиваться ни одной из частей волн с или е данного Треугольника.** (AKU-0020) 🔴
> «B качестве общего правила, трендовая линия b-d не должна пробиваться ни одной из частей волн с или е данного Треугольника
»
*Источник: neely-mwe-1990, стр. 5-33*

```
Когда: pattern == 'triangle'
Правило: not breaks(wave_c, trendline(end(b), end(d))) AND not breaks(wave_e, trendline(end(b), end(d)))
```

3. **В Зигзаге с-волна не должна касаться параллельной трендовой линии (проведённой параллельно линии 0-В через завершение волны-а). Если наблюдается касание, это свидетельствует, что данный зигзаг является частью более сложной Коррекции.** (AKU-0024) 🔴
> «Если фигура, с которой вы работаете, Зигзаг, то с-волна может пробивать параллельную трендовую линию или оставаться далеко от нее, но она не должна ее касаться. Если же наблюдается касание, это свидет...»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'zigzag'
Правило: not touches(wave_c, parallel_line(through=end(wave_a), parallel_to=trendline(start(0), end(B))))
```

4. **Первое подтверждение импульсного характера завершившейся волны требует, чтобы период времени, прошедшего до пересечения последующей волной линии тренда 2-4, не превышал длительности волны 5. В противном случае либо волна 5 развивается в Терминальную, либо волна 4 ещё не завершена, либо гипотеза об импульсном характере конфигурации неверна.** (AKU-0031) 🔴
> «Первое подтверждение импульсного характера завершившейся волны получено, если период времени, прошедшего до пересечения последующей волной линии тренда 2-4, не превышает длительности волны 5. B против...»
*Источник: neely-mwe-1990, стр. 6-1*

```
Когда: pattern == 'impulse' AND position == 'post_pattern'
Правило: duration(until_break(trendline(end(2), end(4)))) <= duration(wave_5)
```

5. **Только четыре из пяти сегментов фигуры (имеющей 6 возможных точек касания одного и того же Порядка) могут одновременно касаться двух противоположных линий тренда. Это правило применяется к Импульсным фигурам (Трендовым и Терминальным) и Треугольникам. Если больше четырёх точек касания одного и того же Порядка с параллельными линиями тренда — формируется не Импульс, а Коррекция.** (AKU-0052) 🔴
> «только четыре из пяти сегментов фигуры (имеющей 6 возможных точек касания "одного и того же Порядка" (т. e. крайних точек волн одной и той же степени)) могут одновременно касаться двух противоположных...»
*Источник: neely-mwe-1990, стр. 9-1*

```
Когда: pattern IN ('impulse', 'terminal_impulse', 'triangle')
Правило: count(segments simultaneously touching both opposite_trendlines) <= 4
```

6. **Никакая часть третьей волны Импульса никогда не должна пробивать линию тренда 2-4. Если пятая волна пересекает линию тренда 2-4, значит, формируется Терминальная Импульсная фигура.** (AKU-0054) 🔴
> «Никакая часть третьей волны Импульса никогда не должна пробивать линию тренда 2-4, а какая-либо часть его пятой волны пересекает эту линию крайне редко. Если пятая волна пересекает линию тренда 2-4, з...»
*Источник: neely-mwe-1990, стр. 9-5*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(end(2), end(4)))
```

7. **Ни одна часть волны-3 не должна пробивать настоящую линию тренда 0-2 в Импульсной фигуре. Если повышение после предполагаемой волны-2 не имеет силы остаться выше линии тренда 0-2, то оно по своему характеру не Импульсное и не может быть частью Импульсной волны-3.** (AKU-0107) 🔴
> «ни одна часть волны-3 не должна пробивать настоящую линию тренда 0-2. Если повышение, удаляющееся от волны-2, не имеет силы остаться выше "первоначальной" линии тренда 0-2, то оно по своему характеру ...»
*Источник: neely-mwe-1990, стр. 12-2*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(start(0), end(2)))
```

8. **Ни одна часть волны-3 или волны-5 не должна пробивать линию тренда 2-4, если только 5-я волна не является Терминальной фигурой.** (AKU-0108) 🔴
> «ни одна часть волны-З или волны-5 не должна пробивать линию тренда 2-4, если только 5-я волна не является Терминальной фигурой.
»
*Источник: neely-mwe-1990, стр. 12-4*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(end(2), end(4))) AND (not breaks(wave_5, trendline(end(2), end(4))) OR subtype == 'terminal')
```

9. **Как только 5-я волна завершится, рынок должен сразу (в течение периода времени, не превышающего длительность волны-5) пробить линию тренда 2-4 и откатиться на большую часть или на всю величину волны-5. Если рынок не удовлетворяет этим требованиям, то линия тренда 2-4 построена неправильно или 5-я волна развивается в Терминальную фигуру.** (AKU-0109) 🔴
> «Как только 5-я волна завершится, рынок должен сразу (в течение периода времени, не превышающего длительность волны-5) пробить линию тренда 2-4 и откатиться на большую часть или на всю величину волны-5...»
*Источник: neely-mwe-1990, стр. 12-4*

```
Когда: pattern == 'impulse' AND position == 'post_pattern'
Правило: breaks(next_move, trendline(end(2), end(4))) AND duration(until_break) <= duration(wave_5) AND retrace(next_move) >= 0.99 * length(wave_5)
```

10. **Чтобы подтвердилась Неудача 5-й волны, откат на величину всей Импульсной фигуры должен произойти быстрее времени её формирования. Чтобы превысить конец Неудавшейся 5-й волны, рынку должно потребоваться по меньшей мере в два раза больше времени, чем заняло формирование всей Импульсной фигуры (1-5).** (AKU-0110) 🔴
> «Чтобы подтвердилась Неудача 5-й волны, откат на величину всей данной Импульсной фигуры должен произойти быстрее времени его (Импульса) формирования. Чтобы превысить конец Неудавшейся 5-й волны, рынку ...»
*Источник: neely-mwe-1990, стр. 12-5*

```
Когда: subtype == 'failure_5th' AND position == 'post_pattern'
Правило: duration(retrace_whole_impulse) < duration(impulse) AND duration(to_exceed_end(failed_5th)) >= 2 * duration(impulse)
```

11. **Построение линии тренда 0-B через волну-е Треугольника (когда волна-b является Треугольником) недопустимо, если это приведёт к её пробитию волной-с. Линию тренда следует проводить через волну-с Треугольника, а не волну-е.** (AKU-0111) 🔴
> «линия тренда "О-В" фактически проведена через волну-с Треугольника, а не волну-е. Построение линии тренда по волне-е привело бы к ее пробитию волной-с, что недопустимо.
»
*Источник: neely-mwe-1990, стр. 12-5*

```
Когда: position == 'wave_b' AND type(wave_b) == 'triangle'
Правило: trendline(0, B) drawn_through end(wave_c) NOT end(wave_e)
```

12. **Волна-е при Треугольной волне-b не должна превышать концевую волну-с Треугольника, даже при «ложном» пробитии линии тренда 0-B. «Ложное» пробитие должно быть очень краткосрочным (относительно).** (AKU-0112) 🔴
> «Это "ложное" пробитые должно быть очень краткосрочным (относительно), и волна-е не должна превышать концевую волну-с Треугольника.
»
*Источник: neely-mwe-1990, стр. 12-6*

```
Когда: position == 'wave_b' AND type(wave_b) == 'triangle'
Правило: not exceeds(end(wave_e), end(wave_c))
```

13. **Для подтверждения Терминальной фигуры активность после волны-(5) должна вернуться к минимуму волны-(4); если этого не происходит, интерпретация неправильная. В 99% случаев рынок будет возвращаться к началу волны-(1) или ниже.** (AKU-0113) 🔴
> «для подтверждения ваших аналитических выводов активность после волны-(5) должна вернуться к минимуму волны-(4); если этого не происходит, ваша интерпретация неправильная.
»
*Источник: neely-mwe-1990, стр. 12-8*

```
Когда: pattern == 'terminal_impulse' AND position == 'post_pattern'
Правило: reaches(next_move, extreme(wave_4))
```

14. **При Растянутой 3-й волне в Импульсной фигуре, независимо от того, какие точки касания использованы для построения трендовых линий, эти линии всегда должны быть параллельны или почти параллельны.** (AKU-0114) 🔴
> «Независимо от того, какие точки касания использованы для построения трендовых линий, эти линии всегда должны быть параллельны или почти параллельны
»
*Источник: neely-mwe-1990, стр. 12-10*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: parallel(channel_lines) == true OR nearly_parallel(channel_lines) == true
```

15. **Если канал Зигзага развивается с расширением (волна-с выходит за параллельную линию), то волна, последующая сразу за этим Зигзагом, не должна откатиться до начального уровня Зигзага. Если эта волна откатывается менее чем на 61.8%, её следует обозначить x-волной, разделяющей две Стандартные Коррекции.** (AKU-0115) 🔴
> «Если канал Зигзага развивается так, как на Диаграмме C, то волна, последующая сразу за ним, не должна откатиться до начального уровня Зигзага. Если эта волна откатывается от Зигзага менее чем на 61.8%...»
*Источник: neely-mwe-1990, стр. 12-14*

```
Когда: pattern == 'zigzag' AND wave_c beyond parallel_line AND position == 'post_pattern'
Правило: not reaches(next_wave, start_level(zigzag))
```

## Условные правила

1. **Все Стандартные Коррекции (кроме Треугольника) характеризуются четырьмя возможными точками касания, но только три из них могут действительно касаться параллельных линий тренда. Если четыре точки одинакового Порядка касаются параллельных линий тренда Стандартной Коррекции (за исключением Треугольника), значит, эта Коррекция — часть Сложной Нестандартной конфигурации.** (AKU-0055) 🟡
> «Все Стандартные Коррекции характеризуются четырьмя возможными точками касания, но только три из них могут действительно касаться параллельных линий тренда. Если четыре точки одинакового Порядка касают...»
*Источник: neely-mwe-1990, стр. 9-2*

```
Когда: pattern IN corrective AND pattern != 'triangle'
Правило: count(same_degree_points touching parallel_trendlines) <= 3
```
