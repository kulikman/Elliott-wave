## Обязательные правила

1. **Правило подобия и баланса: две смежные волны считаются подобными (и могут быть одного Порядка) только если длина меньшей составляет не менее одной трети длины большей — по цене ИЛИ по времени. Если ни одно условие не выполнено — волны принадлежат разным Порядкам и не могут быть объединены напрямую.** (AKU-0003) 🔴
> «Две смежные волны считаются подобными во временном отношении, если длительность меньшей составляет как минимум треть протяжённости большей. Достаточно выполнения хотя бы одного условия — цена ИЛИ врем...»
*Источник: neely-mwe-1990, стр. 4-6*

```
Когда: checking if wave_a and wave_b are same degree
Правило: (min(price(w_a), price(w_b)) >= max(price(w_a), price(w_b)) / 3)
OR
(min(time(w_a), time(w_b)) >= max(time(w_a), time(w_b)) / 3)
```

2. **При формировании импульсной мультиволны одна и только одна из трех нечетных волн (волна 1, 3 или 5) должна быть поливолной. Две остальные должны быть моноволнами.** (AKU-0042) 🔴
> «Одна и только одна из трех нечетных волн в Импульсной фигуре (волна 1, 3 или 5) должна быть поливолной. Две остальные должны быть моноволнами.
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND degree == 'multiwave'
Правило: count(is_polywave in [wave_1, wave_3, wave_5]) == 1 AND count(is_monowave in [wave_1, wave_3, wave_5]) == 2
```

3. **При формировании импульсной мультиволны как минимум одна из коррективных фаз Импульса (волна 2 или 4) должна быть поливолной. Другая может быть как моно-, так и поливолной.** (AKU-0043) 🔴
> «Как минимум одна из коррективных фаз Импульса (волна 2 или 4) должна быть поливолной. Другая может быть как моно -, так и поливолной.
»
*Источник: neely-mwe-1990, стр. doc-ch8*

```
Когда: pattern == 'impulse' AND degree == 'multiwave'
Правило: is_polywave(wave_2) OR is_polywave(wave_4)
```
