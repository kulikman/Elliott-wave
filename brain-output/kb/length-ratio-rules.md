# Правила соотношений длин — Knowledge Base

## Обязательные правила

1. **Пятый сегмент импульса никогда не должен быть меньше 38.2% длины четвёртого сегмента (в ценовом отношении). Когда пятый сегмент короче четвёртого, он называется «Неудавшейся» волной.** (AKU-0006) 🔴
> «Пятый сегмент будет почти, всегда длиннее четвертого, но никогда не должен быть меньше 38.2% длины четвертого (в ценовом отношении).»
*Источник: neely-mwe-1990, стр. 5-1*

```
Когда: # Правило применяется к импульсной структуре, когда определены сегменты 4 и 5
pattern == 'impulse' AND wave_5.is_completed
Правило: # Длина сегмента в ценовом отношении (вертикальное расстояние)
price_length_wave5 = abs(wave_5.end - wave_5.start)
price_length_wave4 = abs(wave_4.end - wave_4.start)

# Mandatory: пятый сегмент никогда не должен быть меньше 38.2% длины четвёртого
price_length_wave5 >= 0.382 * price_length_wave4

# Дополнительная классификация (информационная, не constraint):
# IF price_length_wave5 < price_length_wave4 THEN wave_5.label = "failure" (Неудавшаяся волна)
```

2. **Волна-а Зигзага не должна откатиться от предыдущей Импульсной волны более высокого Порядка (если такая волна имеется) больше, чем на 61.8%.** (AKU-0014) 🔴
> «Волна-а не должна откатиться от предыдущей Импульсной волны более высокого Порядке (если такая волна имеется) больше, чем на 61.8%
»
*Источник: neely-mwe-1990, стр. 5-22*

```
Когда: # Текущий паттерн — Зигзаг (Zigzag)
# Рассматривается волна-a этого Зигзага
# Существует предыдущая Импульсная волна более высокого Порядка (degree),
# частью коррекции которой является данный Зигзаг
pattern == 'zigzag' AND current_wave == 'a'
AND exists(previous_higher_degree_impulse_wave)
Правило: # Волна-а Зигзага не должна откатить более 61.8% от предыдущей
# Импульсной волны более высокого Порядка
#
# previous_impulse = предыдущая Импульсная волна более высокого Порядка
# retracement = насколько волна-a откатила от этой импульсной волны
#
# Длина предыдущей импульсной волны:
prev_impulse_length = abs(end_price(previous_higher_degree_impulse_wave) - start_price(previous_higher_degree_impulse_wave))

# Длина отката волны-a (волна-a идёт в направлении, противоположном предыдущему импульсу):
wave_a_length = abs(end_price(wave_a) - start_price(wave_a))

# Ограничение:
wave_a_length <= 0.618 * prev_impulse_length
```

3. **Волна-с Зигзага должна двигаться (пусть даже ненамного) дальше конечной точки волны-а.** (AKU-0016) 🔴
> «Волна-с должна двигаться (пусть даже ненамного) дальше конечной точки волны-а
»
*Источник: neely-mwe-1990, стр. 5-23*

```
Когда: pattern == 'zigzag'
Правило: # Волна-с должна двигаться дальше конечной точки волны-а
# Для бычьего зигзага (направление вверх): конец волны-с выше конца волны-а
# Для медвежьего зигзага (направление вниз): конец волны-с ниже конца волны-а

if direction == 'up':
    end(wave_c) > end(wave_a)
elif direction == 'down':
    end(wave_c) < end(wave_a)
```

4. **В Плоской коррекции, когда волна-b превышает 138.2% волны-а, нет ни малейшего шанса, что волна-с превысит начальный уровень волны-b (это возможно в случае с-волны Треугольника, но не в случае с-волны Плоской).** (AKU-0017) 🔴
> «Когда волна-Ь превышает 138.2% волны-а, нет ни малейшего шанса, что волна-с превысит начальный уровень волны-Ь (это возможно в случае с-волны Треугольника, но не в случае с-волны Плоской).
»
*Источник: neely-mwe-1990, стр. 5-21*

```
Когда: pattern == 'flat' AND length(wave_b) > 1.382 * length(wave_a)
Правило: not exceeds(end(wave_c), start_level(wave_b))
```

5. **Расширяющиеся Треугольники не могут возникать в b-волнах Зигзагов или в волнах b, с, d более крупных Треугольников.** (AKU-0021) 🔴
> «Расширяющиеся Треугольники не могут возникать в b-волнах Зигзагов или в волнах b, с, d более крупных Треугольников.
»
*Источник: neely-mwe-1990, стр. 5-41*

```
Когда: pattern == 'expanding_triangle'
Правило: position NOT IN {zigzag.wave_b, triangle.wave_b, triangle.wave_c, triangle.wave_d}
```

6. **Волна-с Усечённого Зигзага не может быть короче 38.2% волны-а, но должна быть короче 61.8% волны-а.** (AKU-0022) 🔴
> «Волна-с не может быть короче 38.2% волны-а, но должна быть короче 61.8% волны-а.
»
*Источник: neely-mwe-1990, стр. 5-26*

```
Когда: pattern == 'zigzag' AND subtype == 'truncated'
Правило: 0.382 * length(wave_a) <= length(wave_c) AND length(wave_c) < 0.618 * length(wave_a)
```

7. **Волна-а и волна-b Зигзага должны отличаться по цене. Волна-b составит 61.8% или менее волны-а.** (AKU-0029) 🔴
> «Волна-а и волна-b Зигзага должны отличаться по цене. Волна-b составит 61.8% или менее волны-а.
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'zigzag'
Правило: length(wave_b) <= 0.618 * length(wave_a) AND length(wave_b) != length(wave_a)
```

8. **Временные длины (длительности) трёх последовательно расположенных (смежных) волн одного и того же Порядка не могут быть равны между собой.** (AKU-0053) 🔴
> «временные длины (длительности) трех последовательно расположенных (смежных) волн (одного и того же Порядка) не могут быть равны между собой
»
*Источник: neely-mwe-1990, стр. 9-3*

```
Когда: three consecutive adjacent waves of same degree
Правило: not (duration(w_i) == duration(w_i+1) AND duration(w_i+1) == duration(w_i+2))
```

9. **Ни при каких обстоятельствах три стороны одинакового порядка в составе Треугольника не могут быть равны по цене (плюс/минус 5%).** (AKU-0090) 🔴
> «Ни при каких обстоятельствах три стороны (одинакового порядка) в составе Треугольника не могут быть равны по цене (плюс/минус 5%).
»
*Источник: neely-mwe-1990, стр. 11-19*

```
Когда: pattern == 'triangle'
Правило: not (any 3 of [a,b,c,d,e] are equal within 5% by price)
```

10. **В Горизонтальном Сужающемся Ограничивающем Треугольнике волна-d должна быть короче волны-с, и волна-е должна быть короче волны-d. Волна-b не должна быть менее 38.2% или более 261.8% волны-а.** (AKU-0098) 🔴
> «Волна-d должна быть короче волны-с и волна-е должна быть короче волны-d... волна-Ь не должна быть менее 38.2% или более 261.8% волны - а.
»
*Источник: neely-mwe-1990, стр. 11-22*

```
Когда: pattern == 'horizontal_contracting_limiting_triangle'
Правило: length(wave_d) < length(wave_c) AND length(wave_e) < length(wave_d) AND 0.382 * length(wave_a) <= length(wave_b) <= 2.618 * length(wave_a)
```

11. **При Растянутой 1-й волне волна-3 не должна быть более 61.8% волны-1.** (AKU-0116) 🔴
> «волна-3 не должна быть более 61.8% волны-1.»
*Источник: neely-mwe-1990, стр. 12-26*

```
Когда: pattern == 'impulse' AND extended_wave == 1
Правило: length(wave_3) <= 0.618 * length(wave_1)
```

12. **При Растянутой 3-й волне волна-1 не должна быть связана с волной-3 Внутренним соотношением 61.8%.** (AKU-0117) 🔴
> «Когда в фигуре Растянута волна-3, волна-1 не должна быть связана с волной-3 Внутренним соотношением 61.8%.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: length(wave_1) != 0.618 * length(wave_3)
```

13. **При Растянутой 3-й волне волна-5 должна быть меньше 61.8% Растянутой волны-3.** (AKU-0118) 🔴
> «Волна-5 должна быть меньше 61.8% Растянутой волны-З.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: length(wave_5) < 0.618 * length(wave_3)
```

14. **C-волна Плоской не должна превышать 138.2% а-волны. Если с-волна больше 138.2% волны-а, эта фигура попадает в категорию Удлинённой Плоской.** (AKU-0119) 🔴
> «C-волна Плоской не должна превышать 138.2% а-волны»
*Источник: neely-mwe-1990, стр. 12-31*

```
Когда: pattern == 'flat'
Правило: length(wave_c) <= 1.382 * length(wave_a)
```

15. **Волна-с Зигзага Внутренне не должна быть меньше 61.8% волны-а. Если она меньше, это вероятно связано с Треугольной b-волной, и данный Зигзаг вероятно является стороной Треугольника или частью сложной коррекции, являющейся стороной Треугольника.** (AKU-0120) 🔴
> «Волна-с Внутренне не должна быть меньше 61.8% волны-а.»
*Источник: neely-mwe-1990, стр. 12-30*

```
Когда: pattern == 'zigzag'
Правило: length(wave_c) >= 0.618 * length(wave_a)
```

16. **При Растянутой 3-й волне, если волна-4 — сложная Коррекция, то 5-я волна не должна быть равна или больше волны-1. Если волна-2 — сложная Коррекция, то волна-5 должна быть равна или меньше волны-1.** (AKU-0124) 🔴
> «Если волна-4 сложная Коррекция, то 5-я волна не должна быть равна или больше волны-1. Если волна-2 сложная Коррекция, то волна-5 должна быть равна или меньше волны-1.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: (is_complex(wave_4) IMPLIES length(wave_5) < length(wave_1)) AND (is_complex(wave_2) IMPLIES length(wave_5) <= length(wave_1))
```

17. **При Растянутой 5-й волне волна-5 должна быть не меньше более короткого из двух расстояний — 100% длины (волна-1 + волна-3) или 161.8% волны-3. Максимальная длина волны-5 составляет примерно 261.8% длины (волна-1 + волна-3).** (AKU-0125) 🔴
> «Волна-5 должна быть не меньше более короткого из двух расстояний -100% длины (1+3) или 161.8% волны-3. Она может составлять до 261.8% длины (1+3), но это примерно максимальный предел для длины Растяну...»
*Источник: neely-mwe-1990, стр. 12-28*

```
Когда: pattern == 'impulse' AND extended_wave == 5
Правило: length(wave_5) >= min(1.000 * (length(wave_1) + length(wave_3)), 1.618 * length(wave_3)) AND length(wave_5) <= 2.618 * (length(wave_1) + length(wave_3))
```
