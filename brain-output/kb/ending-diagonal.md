## Обязательные правила

1. **Ценовые проекции длин волн 2 и 4 Терминальных Импульсов должны частично перекрываться. Это обязательное условие для Терминального Импульса.** (AKU-0010) 🔴
> «ценовые проекции длин волн 2 и 4 Терминальных Импульсов должны частично перекрываться»
*Источник: neely-mwe-1990, стр. 5-5*

```
Когда: # Правило применяется к Терминальным Импульсам (Ending Diagonals)
pattern == 'terminal_impulse'  # aka ending_diagonal, structure 3-3-3-3-3
Правило: # "Ценовые проекции длин волн 2 и 4" означают вертикальные (ценовые) диапазоны,
# покрываемые волнами 2 и 4. Эти диапазоны должны частично перекрываться.

# Определяем ценовой диапазон волны 2
wave2_high = max(price_at_start(wave_2), price_at_end(wave_2))
wave2_low  = min(price_at_start(wave_2), price_at_end(wave_2))

# Определяем ценовой диапазон волны 4
wave4_high = max(price_at_start(wave_4), price_at_end(wave_4))
wave4_low  = min(price_at_start(wave_4), price_at_end(wave_4))

# Диапазоны должны частично перекрываться (overlap > 0)
overlap = min(wave2_high, wave4_high) - max(wave2_low, wave4_low)

overlap > 0  # MANDATORY: перекрытие должно существовать
```

2. **Никакая часть третьей волны Импульса никогда не должна пробивать линию тренда 2-4. Если пятая волна пересекает линию тренда 2-4, значит, формируется Терминальная Импульсная фигура.** (AKU-0054) 🔴
> «Никакая часть третьей волны Импульса никогда не должна пробивать линию тренда 2-4, а какая-либо часть его пятой волны пересекает эту линию крайне редко. Если пятая волна пересекает линию тренда 2-4, з...»
*Источник: neely-mwe-1990, стр. 9-5*

```
Когда: pattern == 'impulse'
Правило: not breaks(wave_3, trendline(end(2), end(4)))
```

3. **По завершении Терминальной Импульсной фигуры начальный уровень всей Терминальной фигуры должен быть достигнут за время, не превышающее половины её длительности. Если это правило не выполняется вовсе, гипотеза о Терминальной Импульсной фигуре должна быть отвергнута.** (AKU-0060) 🔴
> «по завершении Терминальной Импульсной фигуры реакция должна быть достаточно мощной, а начальный уровень всей Терминальной фигуры должен быть достигнут за время, не превышающее половины ее длительности...»
*Источник: neely-mwe-1990, стр. 10-1*

```
Когда: pattern == 'terminal_impulse' AND position == 'post_pattern'
Правило: reaches(next_move, start_level(terminal)) AND duration(until_reach) <= 0.5 * duration(terminal)
```

»


5. **Следующая за Терминальным Импульсом волна должна достичь его начального уровня не более чем за 50% времени его формирования; обычно это занимает примерно 25% этого времени.** (AKU-0077) 🔴
> «Следующая за Терминальным Импульсом волна должна достичь его начального уровня не более чем за 50% времени его формирования; обычно это занимает примерно 25% этого времени.
»
*Источник: neely-mwe-1990, стр. doc-ch10*

```
Когда: pattern == 'terminal_impulse' AND position == 'post_pattern'
Правило: reaches(next_wave, start_level(terminal)) AND duration(until_reach) <= 0.5 * duration(terminal)
```

6. **Терминальный Импульс всегда завершает более крупную фигуру, и созданная им точка минимума/максимума должна удержаться примерно в течение как минимум двух периодов времени его формирования (отсчитанных с момента его окончания).** (AKU-0078) 🔴
> «Терминальный Импульс всегда завершает более крупную фигуру, и созданная им точка минимума/максимума должна удержаться примерно в течение как минимум двух периодов времени его формирования (отсчитанных...»
*Источник: neely-mwe-1990, стр. doc-ch10*

```
Когда: pattern == 'terminal_impulse' AND position == 'post_pattern'
Правило: hold(extreme(terminal)) for duration >= 2 * duration(terminal)
```

7. **В Терминальном Импульсе с Растянутой первой волной: волна-2 не должна откатываться более чем на 61.8% волны-1. Волна-3 не должна быть намного больше 61.8%, но и не менее 38.2% волны-1. Волна-5 не может быть более 99% волны-3.** (AKU-0087) 🔴
> «волна-2 не должна откатываться более чем на 61.8% волны-1. Волна-3 не должна быть намного больше 61.8%, но и не менее 38.2% волны-1. Волна-5 не может быть более 99% волны-3»
*Источник: neely-mwe-1990, стр. doc-ch11*

```
Когда: pattern == 'terminal_impulse' AND extended_wave == 1
Правило: retrace(wave_2) <= 0.618 * length(wave_1) AND 0.382 * length(wave_1) <= length(wave_3) <= 0.618 * length(wave_1) AND length(wave_5) <= 0.99 * length(wave_3)
```

8. **Для подтверждения Терминальной фигуры активность после волны-(5) должна вернуться к минимуму волны-(4); если этого не происходит, интерпретация неправильная. В 99% случаев рынок будет возвращаться к началу волны-(1) или ниже.** (AKU-0113) 🔴
> «для подтверждения ваших аналитических выводов активность после волны-(5) должна вернуться к минимуму волны-(4); если этого не происходит, ваша интерпретация неправильная.
»
*Источник: neely-mwe-1990, стр. 12-8*

```
Когда: pattern == 'terminal_impulse' AND position == 'post_pattern'
Правило: reaches(next_move, extreme(wave_4))
```

## Эвристики

- **Причины, приведшие к нарушению одного правила, обычно порождают другое правило. Треугольники и Терминальные фигуры являются исключениями практически из всех правил, стандартных ситуаций и условий.** (AKU-0057) 🟢
> «Причины, приведшие к нарушению одного правила, обычно порождают другое правило.
»
*Источник: neely-mwe-1990, стр. 9-5*

## Исключения

- **В важных точках разворота рынка или при необычных условиях одно из критически важных правил формирования волновой фигуры может не выполняться. Исключительными ситуациями являются: A) завершение мультиволны или более крупной фигуры; B) волна-5 или с-волна Терминальной фигуры; C) волна, являющаяся или завершающаяся Сужающимся или Расширяющимся Треугольником.** (AKU-0056) 🟡
> «Случается, что в важных точках "разворота" рынка или при "необычных" условиях одно из правил, обычно критически важное для правильного формирования волновой фигуры, может не применяться или не выполня...»
*Источник: neely-mwe-1990, стр. 9-5*
