# Соотношения Фибоначчи — Knowledge Base

## Обязательные правила

1. **В Плоской коррекции с нормальной b-волной, волна-с не может составлять менее 38.2% волны-а — это минимальное требование, ниже которого она опуститься не может.** (AKU-0025) 🔴
> «Очень редко волна-с может соотноситься с волной-а с коэффициентом 38.2%, но это минимальное требование (ниже которого она опуститься не может).
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'flat' AND b_type == 'normal'
Правило: length(wave_c) >= 0.382 * length(wave_a)
```

2. **В Плоской коррекции с сильной b-волной (превышающей начальный уровень волны-а), b-волна обычно ограничена соотношением 138.2% или 161.8% волны-а, но не может достигать ни одного из них.** (AKU-0026) 🔴
> «Когда b-волна превышает начальный уровень волны-а, обычно она будет ограничена соотношением 138.2% (это ненастоящее соотношение Фибоначчи, но представляет комбинацию двух - 1.00 и 0.382) или 161.8%, н...»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'flat' AND end_level(wave_b) beyond start_level(wave_a)
Правило: length(wave_b) < 1.618 * length(wave_a) AND length(wave_b) != 1.382 * length(wave_a) AND length(wave_b) != 1.618 * length(wave_a)
```

3. **Сужающиеся Треугольники без соотношений Фибоначчи следует считать невозможными.** (AKU-0027) 🔴
> «Фактически, Сужающиеся Треугольники без соотношений Фибоначчи следует считать невозможными.
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'contracting_triangle'
Правило: has_fibonacci_ratio(among_segments) == true
```

4. **Если в Треугольнике волна-b составляет 61.8% волны-а, то фигура, вероятно, не является Треугольником.** (AKU-0028) 🔴
> «Важное замечание: если волна-b составляет 61.8% волны-а, ваша фигура, вероятно, не является Треугольником.
»
*Источник: neely-mwe-1990, стр. 5-42*

```
Когда: pattern == 'triangle'
Правило: length(wave_b) != 0.618 * length(wave_a)
```

5. **При Растянутой 1-й волне волна-3 не должна быть более 61.8% волны-1.** (AKU-0116) 🔴
> «волна-3 не должна быть более 61.8% волны-1.»
*Источник: neely-mwe-1990, стр. 12-26*

```
Когда: pattern == 'impulse' AND extended_wave == 1
Правило: length(wave_3) <= 0.618 * length(wave_1)
```

6. **При Растянутой 3-й волне волна-1 не должна быть связана с волной-3 Внутренним соотношением 61.8%.** (AKU-0117) 🔴
> «Когда в фигуре Растянута волна-3, волна-1 не должна быть связана с волной-3 Внутренним соотношением 61.8%.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: length(wave_1) != 0.618 * length(wave_3)
```

7. **При Растянутой 3-й волне волна-5 должна быть меньше 61.8% Растянутой волны-3.** (AKU-0118) 🔴
> «Волна-5 должна быть меньше 61.8% Растянутой волны-З.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: length(wave_5) < 0.618 * length(wave_3)
```

8. **C-волна Плоской не должна превышать 138.2% а-волны. Если с-волна больше 138.2% волны-а, эта фигура попадает в категорию Удлинённой Плоской.** (AKU-0119) 🔴
> «C-волна Плоской не должна превышать 138.2% а-волны»
*Источник: neely-mwe-1990, стр. 12-31*

```
Когда: pattern == 'flat'
Правило: length(wave_c) <= 1.382 * length(wave_a)
```

9. **Волна-с Зигзага Внутренне не должна быть меньше 61.8% волны-а. Если она меньше, это вероятно связано с Треугольной b-волной, и данный Зигзаг вероятно является стороной Треугольника или частью сложной коррекции, являющейся стороной Треугольника.** (AKU-0120) 🔴
> «Волна-с Внутренне не должна быть меньше 61.8% волны-а.»
*Источник: neely-mwe-1990, стр. 12-30*

```
Когда: pattern == 'zigzag'
Правило: length(wave_c) >= 0.618 * length(wave_a)
```

10. **При Растянутой 3-й волне, если волна-4 — сложная Коррекция, то 5-я волна не должна быть равна или больше волны-1. Если волна-2 — сложная Коррекция, то волна-5 должна быть равна или меньше волны-1.** (AKU-0124) 🔴
> «Если волна-4 сложная Коррекция, то 5-я волна не должна быть равна или больше волны-1. Если волна-2 сложная Коррекция, то волна-5 должна быть равна или меньше волны-1.»
*Источник: neely-mwe-1990, стр. 12-27*

```
Когда: pattern == 'impulse' AND extended_wave == 3
Правило: (is_complex(wave_4) IMPLIES length(wave_5) < length(wave_1)) AND (is_complex(wave_2) IMPLIES length(wave_5) <= length(wave_1))
```

11. **При Растянутой 5-й волне волна-5 должна быть не меньше более короткого из двух расстояний — 100% длины (волна-1 + волна-3) или 161.8% волны-3. Максимальная длина волны-5 составляет примерно 261.8% длины (волна-1 + волна-3).** (AKU-0125) 🔴
> «Волна-5 должна быть не меньше более короткого из двух расстояний -100% длины (1+3) или 161.8% волны-3. Она может составлять до 261.8% длины (1+3), но это примерно максимальный предел для длины Растяну...»
*Источник: neely-mwe-1990, стр. 12-28*

```
Когда: pattern == 'impulse' AND extended_wave == 5
Правило: length(wave_5) >= min(1.000 * (length(wave_1) + length(wave_3)), 1.618 * length(wave_3)) AND length(wave_5) <= 2.618 * (length(wave_1) + length(wave_3))
```
