# Спринт 0 — Чек-лист визуальной валидации Pine

**Цель:** убедиться что детектор моноволн, классификация П1-7 и распознавание фигур работают на реальных графиках.

**Gate для перехода в Спринт 1:** ≥70% фигур визуально корректны на 8 графиках.

---

## Подготовка TradingView

1. Открой TradingView → Pine Editor
2. Создай новый индикатор → вставь `pine/ewb_monowaves_mtf.pine` → Save → Add to chart
3. Создай второй индикатор → вставь `pine/ewb_confirm.pine` → Save → Add to chart
4. Настройки `ewb_monowaves_mtf.pine` (рекомендуемые для валидации):
   - Порог разворота по ATR: **on**
   - ATR период: **14**
   - ATR множитель: **2.5** (можно увеличить до 3.5 если шумно)
   - Старший ТФ: **240** (если основной 60m), **D** (если основной 240), **W** (если основной D)
   - Показать всё: моноволны, метки, классификацию, фигуры

## Что смотреть на каждом графике

Для каждого графика заполни секцию в `sprint0-report.md`. Конкретные вопросы:

### A. Моноволны (Уровень 1)

- [ ] Моноволны соответствуют визуальным пивотам? (не дробит явный тренд на мелочь, не сливает явные развороты)
- [ ] Метки `m0, m1...` идут последовательно, без пропусков?
- [ ] HTF моноволны (пунктир) совпадают с явными HTF разворотами?
- [ ] Маркер `≠` (правило подобия AKU-0003) появляется в осмысленных местах?

**Рейтинг моноволн:** ⭐⭐⭐⭐⭐ (1-5)

### B. Классификация П1-7 (Уровень 2)

- [ ] На большинстве моноволн есть метка `П1a :5` / `П3b :F3` и т.п.?
- [ ] В трендовом участке преобладают `:5` метки (импульсы)?
- [ ] В коррекциях преобладают `:F3` метки?
- [ ] Метка `?` (Правила 6-7) появляется на необычных откатах?

**Рейтинг классификации:** ⭐⭐⭐⭐⭐ (1-5)

### C. Распознавание фигур (Уровень 3)

Для каждой найденной фигуры на графике:

| Тип        | Кол-во ✅ | Кол-во ❌ | Визуально корректно? |
|------------|-----------|-----------|----------------------|
| Импульс    |           |           |                      |
| Зигзаг     |           |           |                      |
| Плоская    |           |           |                      |
| Треугольник|           |           |                      |
| Двойн.Корр.|           |           |                      |

- [ ] Каналы Нили (0-2, 2-4) рисуются корректно?
- [ ] Цель W5 (161.8%) визуально разумна?
- [ ] Метка Энергии (E+2, E-1) совпадает с типом фигуры?
- [ ] Предупреждения `⚠ t1≈t2≈t3` появляются осмысленно?

**Рейтинг фигур:** ⭐⭐⭐⭐⭐ (1-5)

### D. Баги/странности

Опиши конкретно с барами:
- "Бар 234: Импульс распознан, но W3<W1 — должен быть ❌, помечен ✅"
- "Бар 567: моноволна не нашлась, явный пивот пропущен"
- "Все метки `?` — Определитель Правил даёт мусор"

### E. Скриншоты

Сохрани 2-3 скрина на график:
- Общий вид всего таймфрейма
- Zoom на сомнительную фигуру
- Zoom на хорошо распознанную фигуру

Складывай в `docs/validation/screenshots/sprint0/{symbol}-{tf}-{n}.png`

## Сводный gate

После всех 8 графиков посчитай:

```
total_figures = сумма всех найденных фигур
correct = сумма «визуально корректно» = да
hit_rate = correct / total_figures
```

| hit_rate | Действие |
|----------|----------|
| ≥ 70%    | ✅ Переход в Спринт 1 (HTF bias) |
| 50-70%   | ⚠ Спринт 0.5 — фикс детектора (тюнить ATR множитель, ловить специфичные баги) |
| < 50%    | ❌ Серьёзный пересмотр алгоритма distinguish figures |
