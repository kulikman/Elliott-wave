# wave_books_pool_2 human review queue

Очередь ручной проверки для `wave_books_pool_2`. Этот документ не меняет статусы AKU и не формализует правила; он только группирует то, что Антону нужно проверить перед переводом любых элементов из `draft`.

## Gate

- Не менять `status: draft` автоматически.
- Не менять `formalization.status: not_attempted` автоматически.
- Vision-кандидаты остаются `requires_review`, пока человек не подтвердит трактовку картинки.
- После проверки решать по каждому пункту: `accept`, `revise`, `reject`, `merge_with_existing`, `needs_more_vision`.

## Summary

- Всего pool 2 AKU с `requires_review: true`: 54.
